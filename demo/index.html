Below you will see variations on styling the 'example-box'. I have 6 'example-box' defined in the index.html file. The first 3 are styled by using the attribute in the selector and the second 3 have the same styling applied using class selectors.

I also have an 'example-crate' defined in the index.html file. An 'example-crate' contains the same 6 'example-box's within it. However, the styling on 'example-box' does not get applied to those instances defined within the 'example-crate'

I would like the component user to be able to style the boxes from outside the component (inside the index.html file). I don't want the component user to have to know anything about the component itself other than the published way to style the example-box's within the example-crate.

example-box

Type Attribute Styling

<example-box type="sometype"></example-box> Some Type
<style is="custom-style">
  :root example-box[type=sometype] {
      --example-box-bgcolor: red; <- Different types have different background colors
  }
</style>
